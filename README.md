Welcome to your Tiger Pajamas Web Site!

To run locally:
```
npm install
npm run dev
```

To use the cool custom dividers, put one of the following in your markdown:
```
<hr class="divider-1" />
<hr class="divider-2" />
<hr class="divider-3" />
<hr class="divider-4" />
```

`chickadee.md` contains examples of the custom dividers, as well as buttons:
```
<a href="#" class="solid-button">Solid Button</a>
<a href="#" class="outline-button">Outline Button</a>
```
